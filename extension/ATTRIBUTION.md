# Third-party data attribution

Cí Lens bundles the following third-party data. Its own source code is MIT-licensed
(see ../LICENSE); the data below carries separate licenses that must be preserved on
redistribution.

## CC-CEDICT (`dict/cedict.json`)

Dictionary entries (definitions, pinyin, simplified/traditional forms) are derived from
[CC-CEDICT](https://www.mdbg.net/chinese/dictionary?page=cc-cedict), a community-maintained
Chinese-English dictionary.

- License: [Creative Commons Attribution-ShareAlike 4.0 International (CC BY-SA 4.0)](https://creativecommons.org/licenses/by-sa/4.0/)
- Copyright: CC-CEDICT contributors

## HSK word list (`dict/hsk.json`)

HSK level data is used to classify vocabulary by proficiency level and to support the
"known up to HSK" filtering feature. Levels are derived from
[complete-hsk-vocabulary](https://github.com/drkameleon/complete-hsk-vocabulary)
(new HSK 3.0 word lists).

- License: [MIT](https://github.com/drkameleon/complete-hsk-vocabulary/blob/main/LICENSE)
- Copyright: Yanis Zafirópulos

## CC-Canto (`dict/canto.json`, part of `dict/readings.json`)

Cantonese vocabulary entries and Jyutping readings for CC-CEDICT entries are derived
from [CC-Canto](https://cccanto.org) by Pleco Software.

- License: [CC BY-SA 4.0](https://creativecommons.org/licenses/by-sa/4.0/)

## Wiktionary pronunciation data (part of `dict/readings.json`)

Hokkien (Pe̍h-ōe-jī), Teochew (Peng'im), and supplementary Cantonese (Jyutping)
readings are derived from the English Wiktionary via the machine-readable extraction
at [kaikki.org](https://kaikki.org/) (Tatu Ylonen, Wiktextract).

- License: [CC BY-SA 4.0](https://creativecommons.org/licenses/by-sa/4.0/)
- Copyright: Wiktionary contributors

## Tatoeba example sentences (`dict/sentences.json`)

Example sentences are derived from the `cmn-eng` Mandarin-English sentence pairs
published by the [Tatoeba Project](https://tatoeba.org).

- License: [Creative Commons Attribution 2.0 France (CC BY 2.0 FR)](https://creativecommons.org/licenses/by/2.0/fr/)
- Copyright: Tatoeba Project and its contributors

## Merged Korean-English dictionary (`dict/kodict.json`)

Dictionary entries are merged from three sources:

1. **English Wiktionary** Korean entries, via the machine-readable extraction at
   [kaikki.org](https://kaikki.org/dictionary/Korean/) (Tatu Ylonen: "Wiktextract:
   Wiktionary as Machine-Readable Structured Data", LREC 2022).
   License: [CC BY-SA 4.0](https://creativecommons.org/licenses/by-sa/4.0/) (Wiktionary contributors).
2. **Kengdic** by Joe Speigle ([github.com/garfieldnate/kengdic](https://github.com/garfieldnate/kengdic)).
   License: dual [MPL 2.0](http://www.mozilla.org/MPL/2.0/) / [LGPL 2.0+](https://www.gnu.org/licenses/old-licenses/lgpl-2.0.en.html).
3. **CC-KEDICT** ([github.com/mhagiwara/cc-kedict](https://github.com/mhagiwara/cc-kedict)).
   License: [CC BY-SA 4.0](https://creativecommons.org/licenses/by-sa/4.0/).

Romanization is generated algorithmically (Revised Romanization of Korean) and is not
derived from any dictionary data.

## ms

See the original per-language attribution, reproduced from extension-ms/ATTRIBUTION.md:

# Third-party data attribution

Kamus Lens (Malay reading assistant) bundles dictionary data from the following sources.
Its own source code is MIT-licensed.

## Wiktionary (`dict/dict.json`)

Malay dictionary entries (definitions, Jawi spellings, part of speech) are derived from
the English Wiktionary via the machine-readable extraction at
[kaikki.org](https://kaikki.org/) (Tatu Ylonen: "Wiktextract: Wiktionary as
Machine-Readable Structured Data", LREC 2022).

- License: [CC BY-SA 4.0](https://creativecommons.org/licenses/by-sa/4.0/)
- Copyright: Wiktionary contributors

## Curated cultural and proper-noun set (`data/malay-culture.json`)

A hand-authored set of Malaysian and Malay proper nouns and cultural terms (states,
honorifics, festivals, food, institutions, common given names) is bundled to guarantee
out-of-the-box coverage. These glosses are original to this project.

## Example sentences (`dict/sentences.json`)

Derived from the [Tatoeba Project](https://tatoeba.org) `zsm-eng` Malay–English pairs
(CC BY 2.0 FR). OCR uses Tesseract (Apache-2.0) with the `msa` tessdata_fast model.

## A note on DBP / PRPM

This extension deliberately does **not** bundle data scraped from Dewan Bahasa dan
Pustaka's Pusat Rujukan Persuratan Melayu (prpm.dbp.gov.my). That content is marked
"© Hak Cipta Terpelihara — Dewan Bahasa dan Pustaka" (all rights reserved) and has no
open reuse licence or public API; redistributing it in a shipped extension would be
copyright infringement. If you hold a licence to DBP data, it can be substituted into
`scripts/build-malay.mjs` in place of (or alongside) the Wiktionary source.
